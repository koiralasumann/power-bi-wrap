# Power BI Wrap (No Logo)

A clean HTML wrapper for embedding Power BI dashboards without the Power BI logo/branding.

## Usage

1. Open `power bi wrap no logo.html` in a browser, or host it on a web server.
2. Replace the iframe `src` in the file with your Power BI embed URL.
3. The page provides a minimal header with controls and hides the Power BI footer for a seamless embed.
